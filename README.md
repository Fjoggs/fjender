# fjender
Template render engine for Javascript

## Usage

```js
import { render } from 'fjender'

render('fileName', { mandatory: 'Example mandatory variable' })
```

```html

<html>
    <head>
        <title>{{mandatory}}</title>
    </head>
</html>

```

## Frameworks used
- Ecmascript 6
- Jest
- Eslint with prettier config
- Babel
